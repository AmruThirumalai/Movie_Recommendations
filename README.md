## Project Overview

This project analyzes movie rating data to uncover trends in user preferences and builds a **basic movie recommendation system** using Python. The goal is to demonstrate foundational data analytics and introductory machine learning concepts through a real-world dataset.

---

## Objectives

* Explore and analyze movie ratings data
* Identify trends in genres, ratings, and popularity
* Build a simple movie-to-movie recommendation system
* Visualize and clearly explain insights from the data

---

## Dataset

* **MovieLens Dataset (100k)**
* Includes movie metadata, user ratings, and timestamps
